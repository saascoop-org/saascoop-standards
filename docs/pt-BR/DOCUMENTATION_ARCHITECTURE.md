# Arquitetura da Documentação Institucional

Status da tradução: synced

## Objetivo

Organizar a documentação da SaaSCoop como uma biblioteca institucional trilíngue.

## Idiomas

- `pt-BR`: Português do Brasil
- `en`: Inglês
- `es`: Espanhol

## Estrutura recomendada

```text
docs/
├── pt-BR/
├── en/
└── es/
```

## Princípio

Todo documento institucional relevante deve existir nos três idiomas sempre que possível.

## Versão de referência

O inglês é a versão de referência para interoperabilidade técnica internacional.

Português e espanhol possuem igual importância institucional para a comunidade da SaaSCoop.
