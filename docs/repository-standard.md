# Repository Standard

## Purpose

This document defines the minimum repository standard for SaaSCoop projects.

## Naming

Use lowercase names with hyphens when needed.

Examples:

```text
freireai
empathyai
saascoop-education
ai-engineering-mentor
```

## Required files

Every active repository should include:

- `README.md`
- `LICENSE`
- `CONTRIBUTING.md`
- `CODE_OF_CONDUCT.md`
- `SECURITY.md`
- `ROADMAP.md`
- `.github/ISSUE_TEMPLATE`
- `.github/PULL_REQUEST_TEMPLATE.md`

## Recommended files

- `CHANGELOG.md`
- `docs/`
- `adr/`
- `.github/workflows/`
- `.github/dependabot.yml`
