# Open Source Readiness Checklist

## Repository basics

- [ ] README exists
- [ ] LICENSE exists
- [ ] CONTRIBUTING exists
- [ ] CODE_OF_CONDUCT exists
- [ ] SECURITY exists
- [ ] ROADMAP exists

## Community

- [ ] Issues enabled
- [ ] Discussions enabled when useful
- [ ] Issue forms configured
- [ ] Pull request template configured
- [ ] Maintainers identified

## Automation

- [ ] Markdown lint enabled
- [ ] Link checker enabled
- [ ] YAML validation enabled
- [ ] Dependabot configured when applicable

## Governance

- [ ] Project purpose is clear
- [ ] Relation to SaaSCoop is clear
- [ ] Maturity level is declared
- [ ] Important decisions are documented
