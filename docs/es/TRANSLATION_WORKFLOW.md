# Flujo de Traducción

Estado de traducción: synced

## Objetivo

Mantener sincronizada la documentación trilingüe.

## Estados

- `synced`: traducción alineada con la versión de referencia.
- `needs review`: traducción necesita revisión.
- `draft`: traducción inicial aún en construcción.

## Flujo

1. Crear o actualizar el documento de referencia.
2. Actualizar las versiones PT-BR, EN y ES.
3. Marcar el estado de traducción.
4. Revisar términos sensibles.
5. Abrir un Pull Request.
6. Registrar cambios relevantes en el changelog o roadmap.

## Cuidados

Evitar traducciones literales cuando reduzcan la claridad cultural, jurídica o institucional.
