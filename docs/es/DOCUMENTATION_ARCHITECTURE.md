# Arquitectura de la Documentación Institucional

Estado de traducción: synced

## Objetivo

Organizar la documentación de SaaSCoop como una biblioteca institucional trilingüe.

## Idiomas

- `pt-BR`: Portugués de Brasil
- `en`: Inglés
- `es`: Español

## Estructura recomendada

```text
docs/
├── pt-BR/
├── en/
└── es/
```

## Principio

Todo documento institucional relevante debe existir en los tres idiomas siempre que sea posible.

## Versión de referencia

El inglés es la versión de referencia para la interoperabilidad técnica internacional.

El portugués y el español tienen igual importancia institucional para la comunidad SaaSCoop.
