# Institutional Documentation Architecture

Translation status: synced

## Purpose

Organize SaaSCoop documentation as a trilingual institutional library.

## Languages

- `pt-BR`: Brazilian Portuguese
- `en`: English
- `es`: Spanish

## Recommended structure

```text
docs/
├── pt-BR/
├── en/
└── es/
```

## Principle

Every relevant institutional document should exist in all three languages whenever possible.

## Reference version

English is the reference version for international technical interoperability.

Portuguese and Spanish have equal institutional importance for the SaaSCoop community.
