# Translation Workflow

Translation status: synced

## Purpose

Keep trilingual documentation synchronized.

## States

- `synced`: translation aligned with the reference version.
- `needs review`: translation needs review.
- `draft`: initial translation still in progress.

## Workflow

1. Create or update the reference document.
2. Update PT-BR, EN and ES versions.
3. Mark translation status.
4. Review sensitive terms.
5. Open a Pull Request.
6. Register relevant changes in the changelog or roadmap.

## Cautions

Avoid literal translations when they reduce cultural, legal or institutional clarity.
